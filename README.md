# ❤️ AI Heart Disease Predictor

A smart AI-based web app that predicts heart disease risk using machine learning.

## 🚀 Features
- Predicts Low / Medium / High Risk
- Interactive UI (Streamlit)
- Real-time probability score
- Health factors analysis

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
